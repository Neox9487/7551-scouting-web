# 場記
## 1.前置動作
要跑這個程式，必須先安裝這些程式

1. Node.js (https://nodejs.org/)
2. Python (https://www.python.org/downloads/)
3. Ngrok 需要把網站丟到網路上再安裝 ( https://ngrok.com/)
4. MySQL (其他人的 Windows 安裝教學: https://chwang12341.medium.com/mysql-%E5%AD%B8%E7%BF%92%E7%AD%86%E8%A8%98-%E4%BA%8C-%E4%B8%80%E5%88%86%E9%90%98%E8%BC%95%E9%AC%86%E7%9E%AD%E8%A7%A3%E5%A6%82%E4%BD%95%E5%9C%A8windows%E4%B8%8A%E5%AE%89%E8%A3%9Dmysql-63cce07c6a6c)

## 2.開始執行伺服器
照以下步驟執行

1. 找到資料夾內的 `start.bat` 後雙擊執行  
2. 選擇模式 `4` 取得最新的場次資訊，輸入 `event key` 跟你的 `TPA  API Key`
3. 選擇模式 `1` 或 `2`, 輸入你的 MySQL 帳號密碼

順利的話伺服器就會自己跑起來了
